# TightandHard

Private, 18+ AI companion platform. Subscription SaaS with custom AI girlfriend personas, chat with memory, voice (ElevenLabs), and Stripe billing.

## Stack

- **Framework:** Next.js 13 (App Router) + React + Tailwind + shadcn/ui
- **Auth:** Clerk
- **Database:** MongoDB (Prisma ORM)
- **Payments:** Stripe (subscriptions + webhooks)
- **AI chat:** OpenAI + Replicate (Vicuna / Llama alt)
- **Memory:** Pinecone (vector search over conversation history)
- **Voice:** ElevenLabs (added on top of base template)
- **Images:** Replicate (generated by companion character generator; fallback Cloudinary upload)
- **Rate limiting:** Upstash Redis

## Key features

- Sign up via email, Google, or social login
- Browse AI girlfriend personas by category (Sweet, Sassy, Athletic, Sultry, etc.)
- Chat with any persona — AI responses stream in, conversation memory persists
- Create your own custom persona (Premium tier)
- Monthly subscription with Stripe
- 18+ age gate on first visit
- Full Terms of Service + Privacy Policy pages

## Running locally

Copy `.env.local.example` to `.env` and fill in keys. Then:

```bash
yarn install
npx prisma generate
npx prisma db push
node ./scripts/seed.ts
yarn dev
```

App runs at http://localhost:3000

## Environment variables required

See `.env.local.example` for the full list. Minimum to boot:

- `DATABASE_URL` — MongoDB Atlas connection string
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` + `CLERK_SECRET_KEY`
- `OPENAI_API_KEY`
- `REPLICATE_API_TOKEN`
- `PINECONE_API_KEY`
- `UPSTASH_REDIS_REST_URL` + `UPSTASH_REDIS_REST_TOKEN`
- `STRIPE_API_KEY` + `STRIPE_WEBHOOK_SECRET`
- `ELEVENLABS_API_KEY`

## Legal & compliance

- Site is gated behind a localStorage 18+ age confirmation.
- Terms of Service: `/terms`
- Privacy Policy: `/privacy`
- Adult-content policies require a compliant payment processor; Stripe may require an adult-merchant account or alternative (CCBill, Segpay) depending on final content level.

## Architecture notes

- Base template forked from a public MIT-licensed SaaS AI companion starter.
- Customized for 18+ companion use-case: age gate, rebranded UI, adult-content categories, adjusted Terms.
- Voice integration via ElevenLabs added on top of the base template.
- Character generator (separate repo) produces consistent AI model images that populate new personas.
