# TightandHard — 12 Persona Image Prompts

All prompts produce **hyperrealistic photography, 1:1 aspect ratio (1024×1024)**. Each persona gets a unique build + ethnicity + styling within your spec (90–140 lbs, slim waist, curvy bust/hips). Prompts are written to work in ChatGPT (DALL·E 3), Midjourney v6+, Flux Pro, Replicate SDXL, or any modern image generator.

## How to use

**Option A — ChatGPT/DALL·E:** Paste the prompt into ChatGPT. Add the sentence *"Generate this image."* or use the image generator tool.

**Option B — Use the character-generator app (recommended):** When character-generator is running, create each persona with the Base Prompt field filled in from below. The app feeds it into Replicate with seed locking for consistency.

**Option C — Midjourney:** Add `--ar 1:1 --v 6.1 --style raw` at the end.

## Using these in the character-generator app (recommended path)

1. Open `http://localhost:3001` (once we get that app running)
2. Click "Build your model" → "New character"
3. For each persona, paste the corresponding prompt above into the **"Base prompt"** field at the bottom of the form
4. Fill in name, bio dropdowns, visual style to match
5. Click **"Create character"** → then **"Generate image"** on the detail page
6. Replicate processes it — ~20 seconds per image
7. Copy the resulting Cloudinary URL, update the persona's `src` field in MongoDB

## Using these in ChatGPT

1. Open chat.openai.com
2. Paste any of the 12 prompts
3. ChatGPT (with DALL·E 3) generates the image in ~15 seconds
4. Right-click → Save image
5. Upload to Cloudinary manually, get the URL, paste into the persona's `src` field

Either way works. Option 1 is cleaner because the character-generator keeps the seed locked for consistency across future generations.
