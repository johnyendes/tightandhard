## 🚀 Deployment Instructions

### Backend Deployment
```bash
cd backend
npm install
docker-compose up -d
npm start
```

### Frontend Deployment
```bash
cd frontend
npm install
cp .env.example .env.local
npm run dev
# or
npm run build && npm start
```
